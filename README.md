# StockSight

A Real-Time Stock Market Dashboard built using Python, Streamlit, Plotly and Yahoo Finance.

## Features

- Live stock prices
- Candlestick charts
- Moving averages
- Volume analysis
- Historical data
- Multi-stock comparison

## Technologies Used

- Python
- Streamlit
- Plotly
- Pandas
- yFinance

## Run Locally

pip install -r requirements.txt

streamlit run app.py
