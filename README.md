# 📈 StockSight

Real-Time Stock Market Dashboard built using Python, Streamlit, Plotly, and Yahoo Finance API.

---

## 🚀 Features

- Live Stock Market Data
- Candlestick Charts
- Moving Averages (MA20 & MA50)
- RSI Indicator
- MACD Indicator
- Bollinger Bands
- Portfolio Tracker
- Company Overview
- AI Trend Prediction
- Latest Stock News
- CSV Export
- Multi-Stock Comparison

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- yFinance
- Scikit-Learn
- Feedparser
